# Beibehaltene Cards

## Eigenständige Objekte (korrekter Card-Einsatz)
- `booking-card` — Einzelne Buchungskarte
- `superadmin__module-card` — Modul-Karte mit Toggle
- `superadmin__bg-card` — Hintergrundbild-Preview
- `landing__card` — Modul-Auswahl Landing Page
- `teacher-card` — Mobile Lehrkraft-Listeneintrag
- `cb-counselor-card` — Berater-Auswahlkarte (Buchungsflow)
- `flow-paket-card` — Arbeitspaket-Karte
- `event-card` — Mobile Event-Card

## KPI/Stat Cards
- `teacher-home__stat-card` — Offene Anfragen / Bestätigte Termine
- `flow-stat-card` — Flow-KPI mit Akzentbalken
- `admin-users-stat` — Logins/Admins/Lehrkräfte Zahlen

## Funktionale Container
- `admin-resp-table-container` — Tabellen-Rahmen (responsive)
- `config-section` / `superadmin__section` — Superadmin-Einstellungssektionen
- `sa-card` — DSGVO-Betroffenen-Einträge
- `cb-app` — Öffentliche Buchungsseite (Glasmorphismus)
- `flow-panel` — Flow-Content-Panel (eigener Design-Kontext)
- `ev-workflow` — Event-Workflow-Stepper
- `ev-detail-panel` — Inline-Detailpanel

## Info-Banner (kein Card-Stil)
- `info-banner` — Surface-1-Background ohne Shadow, korrekt eingesetzt
